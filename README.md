<!-- ## ```­¤ōŗ``` Description -->
##

<img src="https://img.shields.io/badge/­¤ōŗ Description-007ACC?style=for-the-badge&labelColor=555555&logoColor=white"
    alt="Description"/>

    Terminal Copy on Select is a lightweight VS Code extension that     automatically copies any text you select in the terminal to your    clipboard. No more right-clicking or using keyboard shortcuts to copy  terminal output!

<!-- ## ```Ō£©``` Features -->
##

<img src="https://img.shields.io/badge/Ō£© Features-007ACC?style=for-the-badge&labelColor=555555&logoColor=white"
    alt="Features"/>

    One-click Copy: Simply select any text in the terminal with your mouse, and it's instantly copied to your clipboard

    Full Selection Support: Works with any amount of text - from a single word to entire log outputs

    No Extra Steps: Eliminates the need for right-click menus or Ctrl+C/Cmd+C shortcuts

    Works with All Terminal Types: Supports integrated terminal, PowerShell, bash, zsh, and more

    Lightweight: Zero performance impact on your VS Code experience

<!-- ## ```­¤Ä»``` How It Works -->
##

<img src="https://img.shields.io/badge/­¤Ä» How It Works-007ACC?style=for-the-badge&labelColor=555555&logoColor=white"
    alt="How It Works"/>

- Open the terminal in VS Code ``(Ctrl+`)``

- `Select` any text `with` your mouse

- The text is `automatically` copied to your `clipboard`

- Paste anywhere (Ctrl+V / Cmd+V)

```That's it! No additional steps required.```

<!-- ## `ŌÜÖ’ĖÅ` Configurations -->
##

<img src="https://img.shields.io/badge/ŌÜÖ’ĖÅ Configurations-007ACC?style=for-the-badge&labelColor=555555&logoColor=white"
    alt="Configurations"/>

This extension works out of the box with zero configuration required. However, if you want to customize its behavior:

```json
{
  // Enable or disable the extension (default: true)
  "terminalCopyOnSelect.enabled": true,
  
  // Show a notification when text is copied (default: false)
  "terminalCopyOnSelect.showNotification": false,
  
  // Preserve formatting when copying (default: true)
  "terminalCopyOnSelect.preserveFormatting": true
}
```
